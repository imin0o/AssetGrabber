# Asset Grabber (images/videos/gifs)

メディアアセットを簡単にスキャンして一括ダウンロードできるChrome拡張機能です。モダンなUIとインテリジェントな重複排除機能を備えています。

## 主な機能

- **高度なスキャン**: 画像 (`img`), 動画 (`video`, `source`), `srcset`, CSSの `background-image`, Aタグのリンクからメディアを自動抽出します。
- **インテリジェントなフィルタリング**: 同名で異なる形式のファイル（例: `.avif` と `.jpg`）がある場合、汎用性の高い形式（PNG > JPG > WEBP > AVIF）を優先してダウンロードし、重複を防ぎます。
- **整理された保存**: 実行ごとに「ドメイン名_日時」の新規フォルダを作成し、その中にアセットを整理して保存します。
- **モダンなUI**: ダークモードを基調としたグラスモーフィズムデザイン。
- **HTML保存**: 最終的なDOM状態のHTMLも同時に保存します。

## インストール方法

1. このリポジトリをダウンロードまたはクローンします。
2. Chromeブラウザで `chrome://extensions/` を開きます。
3. 右上の「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックし、このプロジェクトのフォルダを選択します。

## 使い方

1. アセットを取得したいウェブページを開きます。
2. 拡張機能のアイコンをクリックしてポップアップを開きます。
3. 「ページを調査」ボタンをクリックします。
4. スキャン完了後、「一括ダウンロード」ボタンをクリックすると、ブラウザの標準ダウンロードフォルダ内に新規フォルダが作成され、保存が開始されます。
